# Drupal Terminology

## Entity System
* Content 
  * Stores content in the database
* Config
  * Stores module configuration in a yaml format

### Entity Types
* Content 
    * Articles 
    * Basic Pages
* Taxonomy
  * Tags 
  * Categories
  * Vocabularies
* Users
* Blocks

### Bundles
A collection of fields that belong to a certain entity. Kind of like a sub-type of an entity.

## Fields
* 3 parts
    1. Field Type
        * Tells Drupal what type of content is being stored.
    2. Field Widget
        * Used to enter and edit content into Drupal.
    3. Field Formatter
        * Used to render the stored value to the end user (renders what they see).

## Taxonomy System
Allows you to categorize your content using terms.
* Tags
* Categories
